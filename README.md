<h1>Projeto:</h1>

**1** - Primeiro, clone o repositório:
```bash
$ git clone git@github.com:paulacynthia/atvs-amb-soft.git
```

**2** - No arquivo do projeto, para instalar todas as dependências necessárias:
``` bash
$ npm install 
```

**3** - Para executar os testes:
``` bash
$ npm run test 
```
